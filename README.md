# Luxe Bite Express

## Setup

```sh
git clone <REPO_URL>
cd luxe-bite-express
npm i
npm run dev
```

## Available scripts

- dev: Start Vite dev server
- build: Production build
- preview: Preview built app
- lint: Run eslint
- test / test:watch: Run tests with Vitest
- server: Start the API server (node server/index.js)

## Tech stack

- Vite
- React + TypeScript
- Tailwind CSS + shadcn/ui
